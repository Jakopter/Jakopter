int(* jakopter_frame_processing::callback) (uint8_t *frame, int width, int height, int size)

callback to which is sent every decoded frame.

Parameters
framebuffer containing the raw frame data, encoded in YUV420p. A value of NULL for this parameter means the video stream has ended.
widthframe width
heightframe height
sizesize of the buffer in bytes
Returns
the return value of the callback will be checked by the decoding routine. LESS THAN 0 : the video thread will stop. Anything else : no effect.
