1 # Jakopter
2 ![Jakopter logo](http://jakopter.irisa.fr/wp-content/uploads/2015/06/jakopter3.png)
3 
4 Jakopter is a library which allows to pilot Parrot ARdrone2.0 with a C program or Lua scripts, and with treatments on the drone video feedback.
5 
6 Website : http://jakopter.irisa.fr/ (not available in english yet)
7 
8 ## Requirements
9 ### Mandatory
10 CMake &ge; 2.8
11 C compiler with C99 support (GCC or Clang)
12 
13 ### Optional
14 For Lua bindings : Lua 5.1 or 5.2
15 
16 For video :
17 * FFMPEG (or at least libavcodec) for decoding
18 * SDL2 (for video display)
19 * SDL2_ttf (for text overlay within video display)
20 * SDL2_image (for image overlay within video display)
21 
22 For Leap Motion control : Leap Motion SDK (and a C++ compiler)
23 
24 For direct keyboard control : Curses.
25 
26 
27 ## Build Instructions
28 The project uses CMake as a build system.
29 
30 On UNIX systems you can generate a Makefile with the command:
31 
32  cmake .
33 
34 Then you should be able to compile the project properly with:
35 
36  make
37 
38 You can clear the files generated by CMake with:
39 
40  ./clear_cmake_generated_files
41 
42 ## How To Use
43 You can find the documentation on our main website (http://jakopter.irisa.fr).
